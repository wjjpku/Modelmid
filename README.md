# 项目三：基于文本与逻辑特征的数学作业来源判别（Human vs 各类LLMs）

## 1. 项目背景
随着大语言模型（LLM）的普及，利用 ChatGPT、Kimi、文心一言等工具辅助完成数学理论推导作业或编程作业已成为常见现象。本项目的目标是通过批量采集人类真实作业与不同大语言模型生成的数学题解答，提取风格与逻辑特征，构建分类器，实现对解答来源的精准判别。

## 2. 数据处理说明
为了方便大语言模型 API 批量读取、处理并生成对应的数学题目解答，我们将收集到的原始 LaTeX 题目文件进行了结构化清洗和规范转换。

### 2.1 原始数据
原始数据存放于 `data/` 目录下，包含：
- `ode题目latex代码`（《常微分方程》相关题目）
- `近世代数群论部分题目及解答latex代码（前554行为题目）`（《近世代数》群论相关题目）

### 2.2 处理脚本
我们编写了 `process_data.py` 脚本，该脚本能够：
1. 自动解析并过滤原始 LaTeX 文件中的格式控制标签（如 `\documentclass`、`\begin{document}` 等）。
2. 根据 `\section`、`\subsection`、`\item` 和特定的题号模式（如 `1.1.1.`）智能识别题目层级结构。
3. 将提取的题目及其从属章节、题号等元信息组装，最终输出为结构化的 CSV 文件。

**运行方式**：
```bash
python3 process_data.py
```

### 2.3 输出规范
处理后的文件保存在 `processed_data/processed_data.csv`，共提取了 277 道题目。详细的字段定义与规范请参考 [data_standard.md](data_standard.md)。
